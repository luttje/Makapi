# 🐵 ApiMonkey

This is a simple API testing tool built in C# with WinUI. It's similar to Postman but without the need to log in and with a more straightforward interface. It allows you to send HTTP requests and view the responses in a clean and organized manner. You can save your requests for later use and easily manage them with collections.
